## Overview

Projects allow you to organize your prompts into separate workspaces for better management and team collaboration.

![Project Settings](/images/project-settings.png)

## Project Selection

In the left sidebar, you can see the current project name (e.g., "demo-calendar") at the top. Click on it to:
- Switch between different projects
- Access project management

## Project Settings

Access project settings from the left sidebar to:
- **Change project name**: Edit the project name as needed
- **Invite team members**: Add collaborators to work on the project together

The project management and team collaboration features are currently in development with basic functionality available.
